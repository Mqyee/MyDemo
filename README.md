# MyDemo
海底鱼群效果：共154条鱼，一种是上下或者左右摆动的鱼类，直接使用shader控制；一种是特殊动作的鱼类，如海马、蝠鲼鱼，用烘焙骨骼动画的方式制作。
![image](https://github.com/Mqyee/MyDemo/blob/master/gif/fishs.gif?raw=true) 

植被交互效果：传入角色位置，将植被顶点坐标从局部空间转到世界空间，计算偏移。
![image](https://github.com/Mqyee/MyDemo/blob/master/gif/grass.gif?raw=true)
水面反射效果：将grabpass与水面折射、反射后的采样进行混合得到。
![image](https://github.com/Mqyee/MyDemo/blob/master/gif/water.gif?raw=true)
